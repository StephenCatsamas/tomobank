Phantoms
========

All phantom data sets are generated with `XDesign <http://myxdesign.readthedocs.io/>`_ :cite:`ching:17` 
and  distributed in the data exchange file format :cite:`decarlo:02` together with a python scripts 
to generate them and to load and perform a basic reconstruction using tomoPy  :cite:`Gursoy:14a`.
